QR Code Generation Study
========================

This is a study to understand the generation of dynamic QR codes.

I was originally going to to this from scratch using the specificaiton, however the spec was only available for purchase
for USD$300! So I elected to make use of a library.

This work is intenteded to beintegrated into other prjects that require the use of QR codes.

Code Development
================
Impemented as a windows console application.
